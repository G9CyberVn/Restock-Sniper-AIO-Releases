# RestockSniper AIO — Releases

Public release feed for [RestockSniper AIO](https://github.com/G9CyberVn/Restock-Sniper-AIO).

## Latest

Built artifacts are published under the [Releases](https://github.com/G9CyberVn/Restock-Sniper-AIO-Releases/releases) tab.

Each release ships:
- `RestockSniperAIO.exe` — main Windows x64 binary (obfuscated via garble `-literals`).
- `RestockSniperAIO.exe.sha256` — SHA256 checksum.

`update.bin` at the root of this repo is the encrypted auto-update manifest the running app polls — do not download manually; the app handles upgrades.
